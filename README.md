## Installation

1. **Clone the repository:**
create a folder
open cmd
   ```bash
   git clone  https://github.com/SamiSelx/Sitemap-Builder.git 
    ```

3. **Install dependencies:**
   ```bash
   npm install
   ```


4. **Start the server:**

   1.**testing the simple brute attack:**

      ```bash
   npm run dev
      ```
