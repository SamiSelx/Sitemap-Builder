## Installation

1. **Clone the repository:**
create a folder
open cmd
   ```bash
   git clone  https://github.com/SamiSelx/Sitemap-Builder.git 
    ```

2. **Install dependencies:**
   ```bash
   npm install
   ```


3. **Start the server:**
      ```bash
   npm run dev
      ```
      
4. **Send Post Request with body 'baseUrl': http://localhost:5000/sitemap**
